# FreightMind — Demo 03: Anomaly Detection — Nigeria Air Surge (≤ 2 min)

Seed 42 synthetic Air shipments to Nigeria, watch the stats cache refresh, then see the analytics agent flag the deviation without being told.

---

## 0. Prerequisites

Stack running, browser open at **http://localhost:3000**. Run Demo 02 first to establish baseline silence (recommended).

---

## Step 1 — Baseline query, clean answer (~15 s)

In the **Chat** panel, type:

```
What are the top 5 destination countries by number of Air shipments?
```

**Expected output:** Nigeria at position 4 with 547. No anomaly note in the answer.

**What to point out:**
- Clean response — 547 is just below the IQR upper fence of 565 for Air shipment counts per country

---

## Step 2 — Seed the scenario (~10 s)

In a terminal:

```bash
curl -s -X POST http://localhost:8000/api/demo/seed/nigeria_air_surge \
  | python3 -m json.tool
```

**Expected response:**
```json
{
  "scenario": "nigeria_air_surge",
  "rows_inserted": 42,
  "stats_refreshed": true
}
```

**What to point out:**
- 42 plausible Air shipments inserted (weight 80–600 kg, freight $6k–$22k)
- Stats cache refreshed automatically — the agent's baseline model is now current

---

## Step 3 — Same query, anomaly fires (~20 s)

Type the same question again:

```
What are the top 5 destination countries by number of Air shipments?
```

**Expected output:** Nigeria now at position 3 with 589. The answer contains an anomaly note — e.g.:

> *"Nigeria's Air shipment count (589) is statistically unusual — 4.1× the historical mean of 143, above the typical ceiling of 565. This level of Air dependency may reflect cold chain requirements or port congestion forcing modal shift."*

**What to point out:**
- The answer changed shape — not just the number, but the commentary
- The hypothesis is freight-domain specific, generated by the LLM from the statistical signal
- No alert was configured — the system modelled its own data distribution

---

## Step 4 — Follow the anomaly (~20 s)

Type:

```
Compare average freight cost for Air shipments to Nigeria vs all other Air destinations
```

**Expected output:** Two-row result — Nigeria cost vs rest-of-world average.

**What to point out:**
- SQL disclosure shows `WHERE country = 'Nigeria'` vs `WHERE country != 'Nigeria'`
- The anomaly surfaced a question worth asking; the agent answered it
